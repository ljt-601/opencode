# 文章样式模板说明

本文档介绍微信公众号文章的排版模板和样式规范。

## 可用模板

### 1. Classic（经典风格）

**特点**: 传统、稳重、易读

**适用场景**:
- 技术文章
- 教程指南
- 专业报告

**样式特点**:
- 24px 大标题，深灰色
- 作者信息居左，带分隔线
- 正文 16px，行高 1.8
- 简洁的分段结构

### 2. Modern（现代风格）⭐ 推荐

**特点**: 简约、时尚、美观

**适用场景**:
- 生活方式文章
- 品牌宣传
- 创意内容

**样式特点**:
- 28px 标题，居中对齐
- 字母间距优化
- 正文 15px，行高 2
- 现代化字体栈

### 3. Minimal（极简风格）

**特点**: 干净、优雅、聚焦内容

**适用场景**:
- 文学作品
- 深度思考
- 艺术内容

**样式特点**:
- 使用衬线字体（Georgia）
- 标题不加粗
- 正文 15px，行高 1.9
- 最少的视觉干扰

### 4. Business（商务风格）

**特点**: 专业、活力、品牌感

**适用场景**:
- 企业公告
- 产品介绍
- 商业资讯

**样式特点**:
- 渐变色标题背景
- 白色文字
- 正文微软雅黑
- 视觉冲击力强

### 5. Tech Magazine（科技杂志风格）⭐ AI 快报专用

**特点**: MIT Tech Review 风格，高端杂志感

**适用场景**:
- AI 热点 Top 10 快报
- 科技资讯合集
- 行业动态周报

**样式特点**:
- 深藏蓝 `#1a1a2e` 头部 banner + 渐变色分割条（红→蓝→紫）
- 大号水印编号（01-10）作为卡片背景装饰
- 左侧红色 `#E63946` 强调色条
- 等宽字体编号（SF Mono / Menlo）
- 高级中文排版：PingFang SC + Helvetica Neue，letter-spacing 0.5-4px
- 支持每条新闻独立配图
- 精致页脚带渐隐分割线

**色板**:
- 头部背景: `#1a1a2e`
- 强调色: `#E63946`（MIT 经典红）
- 正文色: `#1a1a2e`
- 二级文字: `#555566`
- 卡片背景: `#F8F9FA`
- 分割线: `#DEE2E6`

## 内容格式化规则

### 标题层级

```markdown
## 主标题（H2）
### 次级标题（H3）
```

### 段落

- 使用空行分隔段落
- 每段不超过 200 字

### 列表

```markdown
- 列表项 1
- 列表项 2
- 列表项 3
```

### 引用

使用特殊样式突出引用内容，适合金句、名言、重点强调。

### 高亮框

使用彩色背景框突出重要信息。

## 排版最佳实践

### 标题规范

- 主标题不超过 20 字
- 次级标题不超过 15 字
- 标题使用动词或疑问句吸引注意力

### 段落规范

- 每段 2-4 句话
- 避免大段文字堆砌
- 段落之间留空行

### 图片使用

- 每篇文章 3-5 张图片为宜
- 图片宽度 900px 最佳
- 封面图 2.35:1 比例

### 颜色搭配

- 标题: #333333（深灰）
- 正文: #2c2c2c（中灰）
- 辅助文字: #666666（浅灰）
- 强调色: #667eea（蓝紫）
- 警告色: #f56565（红色）
- 成功色: #48bb78（绿色）

### 字体规范

- 标题: PingFang SC, Helvetica Neue, Microsoft YaHei
- 正文: -apple-system, BlinkMacSystemFont, Segoe UI
- 英文: Roboto, Helvetica Neue
- 数字: SF Mono, Consolas（用于数据）

## 特殊效果

### 金句样式

使用引用样式突出金句：
```html
<blockquote>金句内容</blockquote>
```

### 数据卡片

使用高亮框展示数据：
```html
<section style="background: #f7fafc; padding: 20px;">
  数据内容
</section>
```

### 分割线

使用 HTML 分割线：
```html
<hr style="border: none; border-top: 1px solid #e2e8f0; margin: 30px 0;">
```

## 移动端优化

- 字体大小不小于 14px
- 行高不小于 1.8
- 段落间距不小于 15px
- 图片宽度 100%
- 避免横向滚动

## 可访问性

- 使用语义化 HTML 标签
- 提供足够的颜色对比度
- 图片添加 alt 属性
- 链接文字清晰明确
