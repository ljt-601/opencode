# 李威记忆法 Skill

基于李威记忆法体系，帮助用户快速记忆任何内容。

## 触发条件

当用户说以下内容时触发此技能：
- "帮我记住..."
- "帮我理解..."
- "怎么记..."
- "如何记忆..."
- "给我讲个记忆方法..."
- "用记忆法记一下..."

## 核心理论

### 三大原理

1. **魔力之七（7±2法则）**
   - 成年人短时记忆容量：7±2个信息单位
   - 应用策略：
     - 长内容分段记
     - 无关联内容尽量关联起来

2. **多维记忆**
   - 人对事物会形成多维度记忆
   - 应用策略：将抽象→形象、动感、有顺序、有特点

3. **艾宾浩斯遗忘曲线**
   - 五个一复习方案：1小时、1天、1周、1月、1季度
   - 应用策略：让知识有联系、有节奏

## 实战技术

### 1. 身体定位记忆法

**15个身体桩子（从上到下）**：
头发 → 眼睛 → 耳朵 → 鼻子 → 嘴巴 → 脖子 → 肩膀 → 胳膊肘 → 手掌 → 肚子 → 屁股 → 膝盖 → 小腿 → 双脚

**适用场景**：
- 记列表（7-15项）
- 记顺序性内容

**案例参考**：
- 56个民族：头发（蒙古族、仫佬族、满族、汉族）...

### 2. 配对联想法

**三种方式**：
- 动作结合：维A缺乏→夜盲症
- 中介物体：空调→美的→美国
- 合并记忆：沙琪玛→满族食品

**适用场景**：
- 记配对关系（国家首都、圣人称号）
- 记因果关系

**案例参考**：
- 菲律宾→马尼拉：你非礼宾客，人家骂你啦
- 苏轼→词圣：吃圣，发明了东坡肉

### 3. 串联记忆法

**三种方式**：
- 直接串联法：脑海中形成画面、空间层次感
- 抽字串联法：抽关键字组词
- 故事串联法：取舍后编成完整故事

**适用场景**：
- 记顺序内容
- 记流程、步骤

**案例参考**：
- 清朝十帝：顺康雍，乾嘉咸，同光宣
- 亚洲流入太平洋的河流：长眉黄龙

### 4. 记忆宫殿

**三个步骤**：
1. 找有顺序的物体（房间物品、身体部位）
2. 内容分组（每组≤7个）
3. 按顺序与物体联系

**适用场景**：
- 记长列表
- 记复杂信息

**案例参考**：
- 十二星座：头发→白羊座（头发白了像羊毛）

### 5. 文字转化技巧

**六种技巧**：
- 以熟记新：用熟悉的知识记新的知识
- 望文生义：看到词联想到画面
- 指代：用符号或图像代替
- 倒叙：雪白→白雪
- 增减字：信用→信用卡
- 谐音转化：最常用，最有效

**适用场景**：
- 记抽象词汇
- 记专业术语

## 工作流程

### Step 1: 分析内容

识别待记内容的类型：
- **列表型**（7-15项）：优先使用身体定位法
- **配对型**（A对应B）：使用配对联想法
- **流程型**（有顺序）：使用串联记忆法
- **长列表型**（>15项）：使用记忆宫殿
- **抽象型**（专业术语）：使用文字转化技巧

### Step 2: 选择技术

根据内容类型选择最合适的记忆技术：
- 列表 ≤ 15项 → 身体定位
- 配对关系 → 配对联想
- 顺序内容 → 串联记忆
- 长列表 → 记忆宫殿
- 抽象概念 → 文字转化

### Step 3: 创建记忆方案

**必须包含的元素**：

1. **形象化描述**（画面感）
   - 描述一个生动的场景
   - 让用户能在脑海中"看到"
   - 例子：脑中想象一个蒙古包样式的帽子包住了木头脑袋

2. **谐音/联想**（连接点）
   - 用中文谐音建立连接
   - 用生活场景类比
   - 例子：菲律宾→非礼宾→骂你啦→马尼拉

3. **记忆口诀**（便于提取）
   - 朗朗上口的句子
   - 押韵或有节奏
   - 例子：顺利治理国家的有康雍乾，嘉道咸，同光宣

4. **逻辑解释**（理解基础）
   - 解释为什么这样记
   - 说明记忆原理
   - 帮助用户理解记忆法本身

### Step 4: 复习计划

基于艾宾浩斯遗忘曲线，给出复习提醒：
- 1小时后快速回顾
- 1天后主动回忆
- 3天后巩固复习
- 1周后应用检验
- 1月后系统回顾

## 输出格式

根据情况灵活调整，但必须包含：

**记忆方案**：

   画面：[生动描述]
   谐音：[谐音解释]

2. 眼睛 - [项目]
   画面：[生动描述]
   谐音：[谐音解释]

- 1天后：从头到尾复述一遍

## 核心原则

1. **本土化优先**：用中国文化场景（满族、沙琪玛、历史人物）
2. **谐音第一**：中文谐音是最有效的记忆工具
3. **画面生动**：描述要能让用户"看到"场景
4. **简单易懂**：记忆法本身要简单，不能比记忆内容更复杂
5. **实用至上**：给出能直接用的方案，不是理论讲解
6. **结合原理**：必要时解释原理，帮助用户理解为什么这样记

## 示例对话

**用户**：帮我记住西湖十景

**助手**：
## 🎯 记忆方案：抽字串联法

**待记内容**：
苏堤春晓、柳浪闻莺、曲院风荷、花港观鱼、雷峰夕照、平湖秋月、南屏晚钟、断桥残雪、双峰插云、三潭映月

**记忆方案**：

**抽字组成口诀**：
苏柳曲，花雷平，南断双三

**画面描述**：
想象这样一个场景：
- **苏柳曲**：苏东坡在柳树下弹曲子
- **花雷平**：花园里突然打雷，湖面变平
- **南断双三**：南边的断桥旁边有两个山峰，水潭映着月亮

**谐音联想**：
- 苏柳曲 → 苏堤春晓、柳浪闻莺、曲院风荷
- 花雷平 → 花港观鱼、雷峰夕照、平湖秋月
- 南断双三 → 南屏晚钟、断桥残雪、双峰插云、三潭映月

**记忆口诀**：
"苏柳弹曲，花园雷平，南边断桥双峰三潭"

**复习计划**：
- 1小时后：闭眼回忆口诀，对应到十景
- 1天后：尝试默写十景
- 3天后：快速复习记不住的
- 1周后：给朋友讲一遍西湖十景的故事

## 注意事项

- **不强制**：如果用户明确说不要某种方法，立即换方法
- **可组合**：多种方法可以组合使用
- **灵活变通**：根据内容特点调整方案
- **用户友好**：记忆方案要符合用户的知识背景和生活经验
- **鼓励主动**：鼓励用户自己创造联想，效果会更好

## 技巧提示

当记忆方案不够好时，可以：
- 换一个谐音
- 换一个场景
- 换一种记忆法
- 分组记忆
- 让用户提供更多背景信息

记住：最好的记忆法是用户自己觉得最顺口、最难忘的方法！
